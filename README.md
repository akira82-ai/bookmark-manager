# 书签管理与智能收藏提示浏览器插件

## 项目简介

书签管理与智能收藏提示是一款功能强大的浏览器扩展插件，旨在帮助用户更高效地管理浏览器书签。它不仅提供了直观的书签组织界面，而且具备智能收藏建议功能，能够基于用户的浏览习惯自动推荐值得收藏的网页，大大简化了书签管理流程，提升浏览效率。

主要功能：
- 书签的树状文件夹结构管理
- 拖拽排序和移动书签
- 智能收藏建议
- 书签搜索和筛选
- URL有效性检查
- 书签导入导出
- 自定义设置和通知

## 安装步骤

### 在Chrome浏览器中加载插件

1. **下载源代码**
   - 将项目代码下载到本地并解压

2. **打开Chrome扩展管理页面**
   - 在Chrome浏览器地址栏输入：`chrome://extensions/`
   - 或通过菜单：`菜单 > 更多工具 > 扩展程序`

3. **启用开发者模式**
   - 在页面右上角找到并开启"开发者模式"

4. **加载已解压的扩展程序**
   - 点击左上角的"加载已解压的扩展程序"按钮
   - 选择项目文件夹所在的位置

5. **确认安装**
   - 安装成功后，浏览器工具栏上会出现插件图标
   - 点击图标即可打开插件弹出界面

## 基本功能验证步骤

### 1. 查看书签

**步骤**：
1. 点击浏览器工具栏中的插件图标，打开弹出界面
2. 检查左侧文件夹树是否显示了所有书签文件夹
3. 选择一个文件夹，右侧应显示该文件夹下的所有书签

**预期结果**：
- 左侧显示完整的书签文件夹树状结构
- 右侧显示选中文件夹下的书签列表
- 书签显示标题和图标

### 2. 添加书签

**步骤**：
1. 在弹出界面中点击"添加书签"按钮
2. 在弹出的对话框中填写书签标题和URL
3. 选择要保存到的文件夹
4. 点击"保存"按钮

**预期结果**：
- 书签成功添加到指定文件夹
- 书签列表自动刷新显示新添加的书签
- 状态栏显示"书签已成功添加"提示

### 3. 搜索书签

**步骤**：
1. 在搜索框中输入关键词
2. 点击"搜索"按钮或按Enter键

**预期结果**：
- 搜索结果显示所有匹配的书签
- 如果没有匹配结果，显示"未找到匹配的书签"提示

### 4. 点击书签打开网页

**步骤**：
1. 在书签列表中选择一个书签
2. 左键点击该书签

**预期结果**：
- 在当前标签页中打开书签URL
- 或者在新标签页中打开书签（取决于设置）

## 高级功能测试步骤

### 1. 智能收藏建议功能

**步骤**：
1. 在浏览器中频繁访问同一网站（至少5次，可在设置中调整阈值）
2. 打开插件主界面，选择过滤器下拉菜单中的"推荐收藏"选项

**预期结果**：
- 显示智能推荐的网站列表
- 包含您多次访问但尚未收藏的网站
- 点击"添加到书签"按钮可快速收藏

### 2. URL有效性检查

**步骤**：
1. 打开插件主界面
2. 点击"检查URL有效性"按钮

**预期结果**：
- 系统开始检查所有书签链接的有效性
- 无效链接会被标记出来（通常以红色或警告图标显示）
- 检查完成后显示结果统计信息

### 3. 书签导入导出

**步骤（导出）**：
1. 在插件主界面的底部点击"导出"按钮
2. 选择保存位置并确认

**预期结果**：
- 生成包含所有书签的HTML或JSON文件
- 显示"导出成功"提示

**步骤（导入）**：
1. 在插件主界面的底部点击"导入"按钮
2. 选择之前导出的书签文件
3. 确认导入

**预期结果**：
- 成功导入书签并显示在相应文件夹中
- 显示导入书签的数量统计

### 4. 拖拽功能

**步骤**：
1. 选择一个书签或文件夹
2. 按住鼠标左键开始拖动
3. 拖到目标位置（另一个文件夹或同一文件夹中的不同位置）
4. 释放鼠标按键

**预期结果**：
- 书签或文件夹成功移动到新位置
- 界面更新以反映新的层次结构

## 功能说明和使用技巧

### 书签管理技巧

1. **使用搜索功能**：当书签数量庞大时，使用搜索功能快速定位特定书签。

2. **文件夹分类**：
   - 为不同类别的书签创建专门的文件夹
   - 考虑按项目、主题或重要性进行分类

3. **利用拖拽功能**：通过拖拽快速整理书签顺序和层级结构。

4. **定期清理**：使用URL有效性检查功能定期清理失效链接，保持书签列表整洁。

### 智能收藏建议

1. **个性化阈值设置**：
   - 在设置中调整访问阈值和天数范围
   - 建议初始设置：访问阈值为5次，天数范围为10天

2. **筛选建议**：使用过滤器下拉菜单中的"推荐收藏"选项查看智能建议。

3. **关闭不需要的通知**：如果不希望收到收藏建议通知，可在设置中关闭。

### 高级操作

1. **批量操作**：按住Ctrl键（Mac上为Command键）可选择多个书签进行批量操作。

2. **键盘快捷键**：
   - Ctrl+F（Mac上为Command+F）：打开搜索框
   - Delete键：删除选中书签
   - F5：刷新书签列表

3. **自定义设置**：
   - 进入设置页面自定义插件行为
   - 可设置默认打开方式、智能建议频率等

4. **数据备份**：养成定期导出书签的习惯，以防数据丢失。

通过掌握这些功能和技巧，您可以充分利用书签管理与智能收藏提示插件，使网页收藏和管理变得更加高效和便捷。
